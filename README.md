Sistema Web de Gestão Acadêmica

Sobre o Projeto
Este projeto consiste no desenvolvimento de um sistema web acadêmico com o objetivo de gerenciar informações de forma estruturada e dinâmica dos projetos de extensão da UFRRJ. A aplicação permite a manipulação de dados por meio de funcionalidades típicas de um sistema CRUD (Create, Read, Update, Delete), simulando um ambiente real de sistemas web.

Tecnologias Utilizadas
- Java (J2EE)
- HTML
- CSS
- Servidor TOMCAT

Funcionalidades
- Cadastro de informações no sistema
- Listagem de dados
- Edição de registros
- Exclusão de registros
- Navegação entre páginas

Estrutura do Projeto
O projeto foi organizado visando separação de responsabilidades e clareza no código, com divisão entre:
Camada de interface (frontend)
Lógica de aplicação
Manipulação de dados

Como Executar
Clone o repositório:
git clone https://github.com/lucasBnunes4/ProjetoWeb2.git
Abra o projeto na sua IDE (VS Code, IntelliJ, etc.)
Execute o servidor/local conforme configuração do projeto
Acesse via navegador
